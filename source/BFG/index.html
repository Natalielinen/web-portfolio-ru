Description

Программа представляет собой графический интерфейс (GUI) для целого ряда консольных распаковщиков игровых ресурсов, основным из которых служит QuickBMS. Так же программа позволяет использовать следующие консольные утилиты: binkconv (для конвертации игрового видео в формате Bink в удобочитаемые большинством плееров), extract (распаковывает архивы нескольких сотен игр на всех версиях движка Unreal Engine), towav (конвертирует аудиофайлы множества игр в популярные форматы, такие как mp3, ogg, wav), offzip (извлекает файлы из любых архивов сжатых алгоритмами Deflate и Zlib), ttarchext (извлекает содержимое архивов всех игр от компании TellTale), SAU (поддерживает распаковку игровых ресурсов многих форматов и конвертацию текстур и спрайтов в PNG) и другие (количество программ приближается к сотне). Помимо всего перечисленного в программе присутствует более 2500 bms-скриптов и 17 wcx-плагинов для Total Commander, среди которых: Total7zip (переделанный в плагин для ТС архиватор 7zip), GAUP (Game Archive Unpacker Plugin — позволяет распаковывать и конвертировать ресурсы из более чем 400 игр), TotalObserver (извлекает содержимое большого количества типов архивов, в том числе игровых, например, игр на движке Source, а также инсталляторов) и других. Все это вместе взятое позволяет в пару кликов распаковать либо конвертировать ресурсы и файлы более чем 10000 приложений и игр (подробный список поддерживаемых игр и форматов можно найти в архиве).

The program is a graphical interface (GUI) for a number of console unpackers of game resources, the main of which is QuickBMS. The program also allows you to use the following console utilities: binkconv (to convert game video in Bink format to readable by most players), extract (unpacks archives of several hundred games on all versions of the Unreal Engine), towav (converts audio files of many games to popular formats such as mp3, ogg, wav), offzip (extracts files from any archives compressed with Deflate and Zlib algorithms), ttarchext (extracts the contents of the archives of all TellTale games), SAU (supports unpacking game resources in many formats and converting textures and sprites to PNG) and others (the number of programs is  approaching a hundred). In addition to all of the above, the program contains more than 2500 bms scripts and 17 wcx plugins for Total Commander, including: Total7zip (7zip archiver converted into a plugin for TS), GAUP (Game Archive Unpacker Plugin allows you to unpack and convert assets from over 400 games), TotalObserver (extracts the contents of a large number of types of archives, including game ones, for example, games on the Source engine, as well as installers) and others. All this taken together allows you to unpack or convert resources and files of more than 10,000 applications and games in a couple of clicks (a detailed list of supported games and formats can be found in the archive).

What's new (version 0.20.03)

НОВОВВЕДЕНИЯ: Теперь практически всем программам, у которых есть возможность упаковки файлов, добавлена поддержка функции запаковки. Достаточно поставить «галочку» на пункте в  интерфейсе «Упаковать», выбрать тип файла, и выбрать нужный файл или папку.

INNOVATIONS: Now almost all programs that have the ability to pack files have added support for the packing function. It is enough to check the box in the «Pack» interface, select the file type, and select the desired file or folder.

САМОЕ ГЛАВНОЕ: Полностью переписан интерфейс, изменен тип интерфейса, благодаря чему, стало возможно передавать функциям параметры непосредственно из интерфейса. Со старым интерфейсом, приходилось писать множество однообразных функций, которые в основном всего лишь назначали скрипт для нужной игры/типа файла, теперь же, все это передается функции непосредственно из интерфейса программы, необходимость в них отпала, и все они были удалены.

Лирическое отступление: Мне следовало использовать этот тип интерфейса с самого начала, но, когда я начал переписывать программу на язык AutoIt (первоначально она была на  Паскале), я не знал этого нюанса, а когда узнал, код программы стал слишком велик для того, чтобы можно было легко взять, и переделать интерфейс полностью. Но, все таки, мои руки дошли до этого, и теперь, код программы уменьшился примерно ВТРОЕ! До версии 0.19, в которой я начал глобальные правки кода, исходный код программы весил около 5 мегабайт, в данной версии, его размер стал менее чем 2 МБ. Мне практически полностью удалось избавиться от программного мусора в коде, оптимизировано и сокращено практически все, что было возможно, и это при том, что кроме правок, я постоянно добавляю в программу новые функции. У программистов есть такое понятие, как «код с запашком», или «дурнопахнущий код», так вот, хочу признаться, из-за неверного выбора типа интерфейса, от моего кода прямо таки «разило вонью»)

IMPORTANT: The interface has been completely rewritten, the interface type has been changed, due to which it became possible to pass parameters to functions directly from the interface. With the old interface, you had to write a lot of monotonous functions that basically just assigned a script for the desired game / file type, but now, all this is transferred to the function directly from the program interface, the need for them has disappeared, and all of them have been removed.

Lyrical digression: I should have used this type of interface from the very beginning, but when I began to rewrite the program into AutoIt (originally it was in Pascal), I did not know this nuance, and when I found out, the program code became too large to be easy take, and remake the interface completely. But I, nevertheless, have done with it, and now, the program code has decreased by about THREE TIMES! Before version 0.19, in which I started global code edits, the source code of the program weighed about 5 megabytes, in this version, its size has become less than 2 MB. I almost completely managed to get rid of software garbage in the code, I optimized and reduced almost everything that was possible, and this despite the fact that, in addition to changes, I constantly add new functions to the program. About not optimized code programmers says: «code smell», and so, I want to admit, due to the wrong choice of the interface type, my previous code really «smelled»).

ОБНОВЛЕНЫ И ДОБАВЛЕНЫ КОМПОНЕНЕТЫ: Обновлены: RAD Video Tools обновлен до версии 4.2e/2020.07 Добавлены: Добавлена программа nspx, для распаковки образов Nintendo Switch.

Руководство

УСТАНОВКА:

  1. Распаковать архив с программой, желательно, чтобы в пути распаковки не было пробелов, символов не на латинице (некоторые компоненты могут из-за этого не работать).
  2. Запустить программу.
  3. При первом запуске программа предложит выбрать папку, в которую будут помещены обработанные файлы, желательно, при ее выборе соблюдать те же правила, что и при установке (см. пункт 1).

ИСПОЛЬЗОВАНИЕ:

  1. Выбрать нужную функцию\игру\тип файла в меню программы.
  2. Выбрать файл.
  3. Дождаться окончания процесса.
  4. Открыть выходную папку (которую вы выбрали в пункте 3 раздела УСТАНОВКА) и посмотреть что получилось.
